# club-AetherFlow
club-AetherFlow is a fresh and modernized skin for rTorrent GUI, ruTorrent. This theme has been featured on projects such as Swizards and has been made exclusively for AetherFlow. Welcome to club-AetherFlow!

## Install

Next, simply clone the club-AetherFlow skin to your themes

```
git clone https://github.com/McEveritts/club-AetherFlow.git club-AetherFlow
chown -R www-data: club-AetherFlow
```
