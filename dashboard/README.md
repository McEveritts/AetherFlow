# AetherFlow Dashboard

The AetherFlow dashboard provides a web-based interface for server management, service control, and system monitoring.

## Features
- Service status monitoring and control
- Bandwidth and network statistics
- Disk usage and system resource monitoring
- Package and plugin management
- Theme and language selection
- User profile and widget preferences
